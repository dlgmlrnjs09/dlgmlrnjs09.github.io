MolraeViewer

개인정보처리방침

최종 업데이트: 2026-04-06

MolraeViewer(이하 "확장프로그램")는 사용자가 디시인사이드 게시판 및 게시글을 스프레드시트 형태의 대체 읽기 화면에서 열람할 수 있도록 돕는 Chrome 확장프로그램입니다.

본 개인정보처리방침은 확장프로그램이 어떤 정보를 처리하는지, 어떤 목적으로 사용하는지, 그리고 어떤 정보를 저장하거나 제3자에게 제공하지 않는지를 설명합니다.

1. 처리하는 정보

확장프로그램은 아래 정보만 기능 수행 범위 내에서 처리합니다.

  • 사용자가 열람 중인 디시인사이드 게시판 또는 게시글 URL
  • 디시인사이드 페이지에서 불러온 게시글 목록, 게시글 본문, 댓글, 이미지/동영상 링크
  • Chrome 확장프로그램 런타임 메시지 처리에 필요한 최소한의 요청 데이터

2. 정보 처리 목적

확장프로그램은 아래 목적에 한해서만 정보를 처리합니다.

  • 디시인사이드 게시판 목록을 스프레드시트 형태로 표시
  • 게시글 상세 내용을 확장프로그램 내부 시트 화면으로 표시
  • 사용자가 명시적으로 요청한 경우 이미지/동영상 보기 기능 제공
  • 새 탭 화면 또는 디시인사이드 페이지 오버레이 모드에서 동일한 대체 읽기 UI 제공

3. 저장 여부

MolraeViewer는 사용자의 개인정보 또는 열람 데이터를 개발자 서버에 저장하지 않습니다.

아래 항목은 수집하거나 영구 저장하지 않습니다.

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 비밀번호, 결제 정보, 인증 정보
  • 디시인사이드 계정 정보
  • 사용자의 전체 브라우징 기록
  • 사용자의 입력 폼 데이터

확장프로그램은 화면 표시를 위해 필요한 데이터를 메모리에서 일시적으로 처리할 수 있으나, 별도의 원격 서버나 자체 데이터베이스로 전송하거나 보관하지 않습니다.

4. 원격 전송 및 제3자 제공

확장프로그램은 기능 수행을 위해 디시인사이드 관련 페이지에 네트워크 요청을 보낼 수 있습니다.

  • 게시판 목록 HTML 조회
  • 게시글 상세 HTML 조회
  • 댓글 데이터 조회
  • 디시인사이드 이미지 보기 페이지 로드

이 과정은 사용자가 현재 열람하려는 디시인사이드 콘텐츠를 표시하기 위한 것이며, 개발자가 운영하는 별도 서버로 데이터를 중계하거나 판매하지 않습니다.

MolraeViewer는 사용자의 개인정보나 열람 데이터를 제3자와 공유, 판매, 대여하지 않습니다.

5. 권한 사용 설명

확장프로그램은 아래 권한을 사용합니다.

tabs

  • 사용자가 Esc 키를 눌렀을 때 현재 탭을 닫는 기능에 사용합니다.

디시인사이드 호스트 권한

  • https://gall.dcinside.com/*
  • https://m.dcinside.com/*

위 권한은 디시인사이드 게시판/게시글 페이지에서 목록 및 상세 데이터를 불러오고, 오버레이 읽기 화면을 제공하기 위해서만 사용합니다.

6. 사용자 선택과 통제

사용자는 언제든지 아래 방법으로 확장프로그램 동작을 중단할 수 있습니다.

  • Chrome 확장프로그램 관리 화면에서 비활성화 또는 삭제
  • 디시인사이드 페이지에서 원본 보기 버튼으로 오버레이 숨김

확장프로그램을 삭제하면 해당 확장프로그램의 실행 권한도 함께 제거됩니다.

7. 아동의 개인정보

MolraeViewer는 아동을 대상으로 설계된 서비스가 아니며, 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

8. 정책 변경

본 개인정보처리방침은 기능 변경, 정책 변경, 법적 요구사항 반영 등을 위해 수정될 수 있습니다. 중요한 변경이 있을 경우 최신 버전을 공개된 문서로 업데이트합니다.

9. 문의

개인정보처리방침 또는 확장프로그램의 데이터 처리 방식과 관련한 문의는 Chrome Web Store 등록 연락처 또는 배포자가 안내하는 공식 문의 채널을 통해 접수할 수 있습니다.
